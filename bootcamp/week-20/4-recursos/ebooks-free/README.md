# Ebooks Gratuitos — Semana 20: LATERAL Joins

## Recursos recomendados

### The Art of PostgreSQL
- **Autor**: Dimitri Fontaine
- **Disponible**: https://theartofpostgresql.com/ (capítulos gratuitos)
- **Relevancia**: Capítulo sobre `LATERAL` y funciones tabulares en contextos reales.

### Learning SQL (3rd Ed.) — Alan Beaulieu
- **Disponible**: O'Reilly / biblioteca pública digital
- **Relevancia**: Fundamentos de subqueries correlacionadas —
  base conceptual para entender `LATERAL`.
