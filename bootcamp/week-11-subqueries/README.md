# Semana 11 — Subqueries

## Tema

Consultas anidadas dentro de otras consultas: subqueries escalares,
subqueries en `WHERE` con `IN`/`EXISTS`, subqueries correlacionadas
y subqueries en `FROM` como tablas derivadas.

## Objetivos

- Escribir subqueries escalares que retornan un único valor
- Filtrar con `IN`, `NOT IN`, `EXISTS` y `NOT EXISTS`
- Usar una subquery en `FROM` como tabla derivada
- Identificar cuándo preferir un JOIN sobre una subquery
