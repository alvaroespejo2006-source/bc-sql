# Semana 12: CTEs y CASE WHEN

## Descripción

Introducción a las Common Table Expressions (CTEs) con la cláusula `WITH`
y al operador condicional `CASE WHEN`. Los CTEs reemplazan subqueries en `FROM`
con código más legible; `CASE WHEN` permite lógica condicional dentro de queries.

## Objetivos de la semana

1. Escribir CTEs simples y encadenados con la cláusula `WITH`
2. Referenciar CTEs en la consulta principal como si fueran tablas
3. Usar `CASE WHEN` para clasificar filas y calcular valores condicionales
4. Combinar CTEs y `CASE WHEN` en un solo reporte analítico
