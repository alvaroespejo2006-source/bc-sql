# Ebooks gratuitos — Semana 12: CTEs y CASE WHEN

## Recomendados

### 1. Learning SQL (Alan Beaulieu) — O'Reilly

El capítulo 14 cubre CTEs (WITH) en detalle: CTEs simples, en cascada
y recursivos. También incluye expresiones condicionales (`CASE WHEN`).

🔗 https://openlibrary.org/books/OL24202440M

### 2. SQL for Data Scientists (Renée M. P. Teate)

El capítulo 8 explica CASE WHEN con aplicaciones prácticas para análisis
de datos: clasificación de segmentos, pivoting y agregación condicional.

🔗 https://sqlfordatascientists.com/

### 3. Mode SQL Tutorial — Advanced SQL

Sección sobre CTEs y CASE, con ejemplos de análisis de negocio sobre
datos reales y diferencias entre CTEs y vistas.

🔗 https://mode.com/sql-tutorial/sql-case/
