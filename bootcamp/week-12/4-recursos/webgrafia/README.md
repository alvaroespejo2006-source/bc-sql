# Webgrafía — Semana 12: CTEs y CASE WHEN

## Referencias web

### 1. PostgreSQL Docs — WITH Queries (CTEs)

Documentación oficial sobre CTEs: sintaxis, CTEs en cascada, CTEs
recursivos y tips de materialización (MATERIALIZED / NOT MATERIALIZED).

🔗 https://www.postgresql.org/docs/current/queries-with.html

### 2. SQLite Docs — Common Table Expressions

Referencia completa del `WITH` en SQLite, incluyendo CTEs recursivos.
Cubre compatibilidad con versiones (CTEs desde SQLite 3.8.3).

🔗 https://www.sqlite.org/lang_with.html

### 3. PostgreSQL Docs — CASE Expression

Referencia sobre las formas "buscada" y "simple" de CASE, con notas
sobre evaluación en cortocircuito y valores NULL.

🔗 https://www.postgresql.org/docs/current/functions-conditional.html
