# Videografía — Semana 12: CTEs y CASE WHEN

## Videos recomendados

### 1. SQL WITH Clause (CTE) — Programming with Mosh (YouTube)

Explicación paso a paso de CTEs simples y en cascada. Compara CTEs
con subqueries en FROM y explica cuándo preferir cada uno.
Duración: ~25 min.

🔗 https://www.youtube.com/watch?v=K1WeoKxLZ5o

### 2. SQL CASE Statement — Socratica (YouTube)

Tutorial claro sobre `CASE WHEN` con ejemplos de clasificación,
agregación condicional y uso dentro de `ORDER BY`.
Duración: ~12 min.

🔗 https://www.youtube.com/watch?v=NeB2SnaYmm0

### 3. Advanced SQL: CTEs and Window Functions — Alex the Analyst (YouTube)

Combina CTEs con otras técnicas avanzadas (window functions).
Muestra pipelines de análisis de datos del mundo real.
Duración: ~40 min.

🔗 https://www.youtube.com/watch?v=G3lJAxg1cy8
