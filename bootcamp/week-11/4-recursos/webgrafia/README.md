# Webgrafía — Semana 11: Subqueries

## Referencias web

### 1. PostgreSQL Docs — Subquery Expressions

Documentación oficial de PostgreSQL sobre EXISTS, IN, ANY, ALL y subqueries
escalares, con definición técnica y comportamiento con NULL.

🔗 https://www.postgresql.org/docs/current/functions-subquery.html

### 2. SQLite Docs — The SELECT Command

Referencia sobre subqueries en FROM, WHERE y SELECT en SQLite,
incluyendo notas de comportamiento específicas del motor.

🔗 https://www.sqlite.org/lang_select.html

### 3. Use The Index, Luke — Subqueries

Explica el impacto de rendimiento de las subqueries correlacionadas
(N+1 problem) y cuándo un JOIN es más eficiente.

🔗 https://use-the-index-luke.com/sql/where-clause/obfuscation/subqueries
