# Ebooks gratuitos — Semana 11: Subqueries

## Recomendados

### 1. Learning SQL (Alan Beaulieu) — O'Reilly

El capítulo 9 cubre subqueries en profundidad: escalares, correlacionadas,
en FROM, en WHERE y en HAVING. Disponible en Open Library.

🔗 https://openlibrary.org/books/OL24202440M

### 2. SQL for Data Scientists (Renée M. P. Teate)

El capítulo 5 explica subqueries con casos prácticos de análisis de datos,
incluyendo subqueries correlacionadas y tablas derivadas.

🔗 https://sqlfordatascientists.com/

### 3. Mode SQL Tutorial — SQL Subqueries

Tutorial gratuito sobre subqueries con ejemplos interactivos y diferencias
entre subqueries correlacionadas y no correlacionadas.

🔗 https://mode.com/sql-tutorial/sql-subqueries/
