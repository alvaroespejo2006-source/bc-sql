# Videografía — Semana 11: Subqueries

### 1. SQL Subqueries Tutorial — Programming with Mosh (YouTube)

Explicación completa de subqueries escalares, correlated subqueries y
tablas derivadas con ejemplos prácticos en tiempo real.
Duración: ~35 min.

🔗 https://www.youtube.com/watch?v=i5acg3Hvu6g

### 2. SQL EXISTS vs IN — Fireship (YouTube)

Comparación clara entre EXISTS e IN, con casos donde cada uno conviene
y la trampa de NOT IN con valores NULL.
Duración: ~10 min.

🔗 https://www.youtube.com/watch?v=4W9mNjgQkmI

### 3. Derived Tables in SQL — Socratica (YouTube)

Tutorial sobre tablas derivadas en FROM con ejemplos paso a paso.
Explica por qué el alias es obligatorio y cómo combinar con JOINs.
Duración: ~15 min.

🔗 https://www.youtube.com/watch?v=HXV3zeQKqGY
